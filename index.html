Research Focus

Research

TWARLeT-2.5D — First-Author Research Project

Reinforcement Learning for 2.5D Chiplet Placement

Designed an end-to-end reinforcement-learning framework for thermally aware 2.5D chiplet placement. The system uses imitation learning, PPO, placement constraints, MILP-based routing, and thermal simulation to explore chiplet layouts under realistic engineering trade-offs.

Developed automated evaluation pipelines across industrial-scale benchmarks, reducing wirelength by up to 27% and achieving 58× faster time to solution than simulated annealing. The work focuses on fast design-space exploration for heterogeneous chiplet systems.

Reinforcement Learning · Imitation Learning · PPO · HotSpot · TAP-2.5D · MILP · Thermal Optimization

Oscillator-Based Ising Optimization

Combinatorial Optimization and Max-Cut-style Problems

Building oscillator-based Ising-machine optimization workflows for graph-based combinatorial problems. The work studies how oscillator dynamics, staged parameter schedules, and graph structure affect solution quality and convergence behavior.

Evaluation focuses on convergence stability, final cut quality, graph size effects, runtime, and repeatability across experimental trials. The implementation uses systems-oriented C++ workflows with parallel computing concepts.

C++ · MPI · OpenMP · Ising Models · Max-Cut · Graph Optimization · Parallel Computing

GNN Freeze-Mask Optimization

Graph Neural Networks for Runtime Reduction

Working on graph neural network methods to predict stable nodes in combinatorial optimization workflows. The goal is to reduce runtime by freezing reliable node decisions while preserving final solution quality on hard graph instances.

Built graph-level and node-level data collection workflows for Gset graph experiments, with evaluation centered on cut preservation, runtime reduction, stable-node selection, and unseen graph generalization.

GNN · PyTorch · Graph Mining · Stable Node Prediction · Optimization · Benchmarking
