Research

Research Work

My research is focused on AI/ML methods for optimization and systems problems. I currently highlight two main research directions: reinforcement learning for 2.5D chiplet placement and oscillator-based Ising-machine optimization.

TWARLeT-2.5D — First-Author Research Project

Reinforcement Learning for 2.5D Chiplet Placement

Designed an end-to-end reinforcement-learning framework for thermally aware 2.5D chiplet placement. The system uses imitation learning, PPO, placement constraints, MILP-based routing, and thermal simulation to explore layouts under realistic engineering trade-offs.

Built automated evaluation pipelines across industrial-scale benchmarks, reducing wirelength by up to 27% and achieving 58× faster time to solution than simulated annealing. The work focuses on faster design-space exploration for heterogeneous chiplet systems.

Reinforcement Learning · Imitation Learning · PPO · Python · PyTorch · Thermal Simulation · MILP · Optimization

Oscillator-Based Ising Optimization

Combinatorial Optimization and Max-Cut-style Problems

Building oscillator-based Ising-machine optimization workflows for graph-based combinatorial problems. The work studies how oscillator dynamics, staged parameter schedules, and graph structure affect solution quality and convergence behavior.

Evaluation focuses on convergence stability, final cut quality, graph size effects, runtime, and repeatability across experimental trials. The implementation uses systems-oriented C++ workflows and parallel computing concepts.

C++ · MPI · OpenMP · Ising Models · Max-Cut · Graph Optimization · Parallel Computing

RL-Based 2.5D Chiplet Placement

Developed a reinforcement learning framework for thermal- and wirelength-aware chiplet placement using placement constraints, thermal simulation, action masking, and routing-aware evaluation.

The environment models chiplet dimensions, power, connectivity, feasible placements, orientation choices, and graph-based state features. The policy uses PPO-style optimization to generate feasible layouts under system constraints.

Built evaluation around peak temperature, wirelength, feasibility, runtime, and comparison against simulated annealing and optimization-based baselines.
